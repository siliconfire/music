Drop image files into this folder to expose them in the board background catalog.

Supported extensions:
- .jpg
- .jpeg
- .png
- .webp
- .avif
- .gif
- .svg

Display name rule:
- Taken from the file name (without extension)
- '-' characters are replaced with spaces

Examples:
- dark-forest.jpg -> "dark forest"
- sunset-skyline.webp -> "sunset skyline"
